## HTML

----

### 常用标签

1. 换行：
   ```html
   <br>
   ```

2. 水平分割线

   ```html
   <hr>
   ```

3. 加粗
   ```html
   <strong></strong>
   <b></b>
   ```

4. 斜体
   ```html
   <em></em>
   <i></i>
   ```

5. 删除线
   ```html
   <del></del>
   <s></s>
   ```

6. 上下标
   ```html
   <sup></sup>
   <sub></sub>
   ```
